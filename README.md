## Data

This project uses the [SMS Spam Collection Dataset](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset).

### How to get the data

**Manual download**:
   - Go to: https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset
   - Log in (via Google Account) and download `spam.csv`
   - Place the file in: `data/raw/spam.csv`
